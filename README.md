# Task Tracker (Vue 3 + Quasar)

## Setup
npm install
quasar dev

## Features
- Add / delete tasks
- Mark completed
- Filter by status
- Persist data in LocalStorage

## Tech Stack
Vue 3, Quasar, TypeScript, Pinia
quasar build
```

See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
